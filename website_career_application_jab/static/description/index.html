Website Career Application

Comprehensive Career Application Form for Odoo 17

Features

Public Website Form

Complete application form accessible at /kariyer/basvuru with comprehensive data collection including personal information, experience, education, and documents.

Backend Management

Full backend interface to manage applications with tree and form views, filtering, and activity tracking.

Key Capabilities

  • Personal information and address collection
  • Passport and disability status tracking
  • Military status (for male applicants)
  • Family reunion with spouse and children (max 2)
  • Language certificates and German level
  • Recognition (Denklik) status tracking
  • Work experience with departments and devices
  • Document uploads with validation (10MB max)
